# Internship Tracker Project

A console-based Internship Tracking System developed using **Python, Object-Oriented Programming (OOP), SQL Server (SSMS), and pyodbc**.

## Features

* Student Management (Add, View, Update, Delete)
* Internship Application Tracking
* Interview Management
* Search Applications by Status
* Generate Application Reports
* SQL Server Database Integration

## Technologies Used

* Python
* OOP
* SQL Server (SSMS)
* pyodbc

## Database Structure

* **Student**
* **Company**
* **Application**
* **Interview**

## How to Run

1. Execute the SQL script in SSMS to create the database and tables.
2. Install dependencies:

```bash
pip install pyodbc
```

3. Update the server name in `database.py`.
4. Run:

```bash
python main.py
```
